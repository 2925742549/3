# GitHub 上传说明

1. 新建仓库，例如 SafeCamFinal
2. 解压 ZIP
3. 上传解压后的所有内容
4. 如果 `.github` 没出现，点 Add file → Create new file
5. 文件名填 `.github/workflows/build-apk.yml`
6. 内容复制根目录 `build-apk.yml`
7. Commit changes
8. 点 Actions → Build SafeCam Final APK → Run workflow
9. 下载 SafeCamFinal-debug-apk，解压得到 app-debug.apk

安装新版前，建议先卸载之前测试的旧版。
